# Contributing

Thank you for your interest in contributing to Desktop Tasks Wallpaper System!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone <your-fork-url>`
3. Create a branch: `git checkout -b feature/your-feature-name`
4. Make your changes
5. Test your changes
6. Submit a pull request

## Development

### Testing

Before submitting a PR, please test:
- The installation script works correctly
- Wallpapers generate without errors
- Both landscape and portrait templates render properly
- Task addition script works as expected

### Code Style

- Use bash best practices
- Add comments for complex logic
- Keep scripts readable and maintainable

## Reporting Issues

Please include:
- macOS version
- Chrome version
- Error messages (if any)
- Steps to reproduce
